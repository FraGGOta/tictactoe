# Tic-tac-toe

## description
  Network tic-tac-toe game application with multiservers supporting
  
## compilation
  ### server
   #### from server dir: 
    make/make all
  ### client
   #### from client dir: 
    make/make all
    
## run program
  ### server
   #### from server dir:
   #### order is important!
    ./bin/tictactoe - run main server
    ./bin/tictactoe --opt_serv_1 - run first optional server
    ./bin/tictactoe --opt_serv_2 - run second optional server
  ### client
   #### from client dir: 
    ./bin/client
